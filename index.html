Ultra ERP

Sistema ERP completo para a empresa Ultra Tecnologias. Com controle de perfis de usuários indo desde clientes, passando por consultores, técnicos, setor financeiro, administradores e super administradores. Controle de pagamentos com API para geração de boletos e pagamentos em cartão integrados no sistema, em suma, módulos para administração, financeiro, estoque, serviços, vendas, contratos, suporte e módulos especiais.

UTR Bus

Sistema para a empresa Ultra Tecnologias utilizado para controle de frotas de ônibus, possibilitando a criação de rotas, monitoramento em tempo real, controle de passageiros, motoristas, veículos, dentre outras ações.

Geração Rádios

Web rádio com player próprio, controle de usuários, sistema para realização de pedidos musicais e interação com a rádio, feita para suportar múltiplas estações e streamers.

DVS BET Apostas Online

Sistema para realização de apostas online, com a possibilidade de realizar simulações com combinações de jogos para depois realizar a aposta e o cadastro de múltiplos campeonatos. O sistema também conta com um bot que alimenta a base de dados de campeonatos, fazendo sempre ter disponíveis uma lista de jogos para cadastrar, para o administrador alterar apenas os valores das apostas pré-definidas.

YouTube Analytics

Software para realização de buscas avançadas no YouTube utilizando da API do mesmo para encontrar canais com boa taxa de prospecção, utilizado por empresas de marketing digital para encontrar influenciadores digitais.
